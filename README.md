steam-wishlist-diff
===================

A simple way to track price fluctuations in Steam wishlist

###Dependencies###

* requests
* bs4
* (xtermcolor)

###Usage###

1. Specify your SteamCommunity account name in script (also can be stored in `account.txt`). Your account has to be public.
2. Enjoy

####CL keys####

 - `--help`
 - `--deletelast`
 - `--moves`
 - `--nowrite`
 - `--salesonly`
